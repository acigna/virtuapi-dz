Classe pour gérer la table etablissement et sa relation avec la table faculte-etablissement.

Classe pour gérer la table etablissement et sa relation avec la table faculte-etablissement. Elle est utilisée pour l'ajout, la modification, la suppression, et la lecture d'éléments à partir de la table etablissement. Elle est aussi utilisée aussi pour l'ajout et la suppression à partir de la table faculte-etablissement, ainsi que la lecture d'éléments de la table faculte par l'intérmediaire de la table faculte-etablissement.

  • see: Faculte
